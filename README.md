# Análisis de Factibilidad: Sistema de Geocerca para Ganado Ovino

Este proyecto documenta la investigación técnica y económica para la implementación de un sistema de collares con Geocerca para ovinos en la Región Sur de Río Negro, Argentina. Desarrollado como **Proyecto Social (PTS)** para la UNRN (2025-2026).

## Problemática Regional
La producción ovina en la estepa patagónica enfrenta desafíos críticos debido a la depredación y la pérdida de animales por factores climáticos. El estudio evalúa la viabilidad de utilizar tecnología IoT para el control y contención del ganado.

## Alcance del Estudio Técnico
Se analizaron y compararon diversas tecnologías para conformar un prototipo de bajo costo y alta eficiencia:
- **Microcontroladores:** Evaluación de ESP32, ESP32-S3 y módulos Heltec LoRa.
- **Geoposicionamiento:** Comparativa de módulos GPS u-blox y QUECTEL considerando precisión y consumo energético.
- **Comunicaciones:** Implementación de protocolos LoRaWAN y LoRa P2P para cubrir grandes extensiones con baja infraestructura.
- **Estímulo Correctivo:** Análisis de circuitos elevadores de voltaje para generar correctivos conductuales ante la salida del perímetro.

## Viabilidad Económica
El informe incluye un desglose detallado de costos de hardware, fabricación de PCB y ensamblaje, comparando la solución propuesta con productos comerciales existentes en el mercado internacional.
